## How to Play
Goal : Find 3 white cats to unlock exit door then escape without be attack by monster

- 'W A S D' to **Move**
- 'F' to use **FlashLight**
- 'LShift' to **Sprint**
- Camera control by mouse
- **FlashLight** can neutralize monster (FlashLight can out of battery)
- **Battery**, **Key** and **White Cat** can be acquired around map
- Some White Cats is inside chest which needed **Key** to be unlock.

## Key Learning

- **Cross-Functional Technical Communication:** Bridged the gap with Level Designers by developing event-driven documentation and node-based mapping to synchronize technical spawn points.
- **AI Navigation:** Gained experience in Unity AI Agents, implementing custom navigation paths and node-based positioning for complex environmental movement.
- **Behavioral Logic Design:** Gained experience State Machines (FSM) to engineer diverse AI behaviors, including Patrolling, Attacking, and Observing.
- **Architectural Efficiency:** Optimized code reusability and reduced system complexity through Singleton Patterns, ensuring a streamlined development workflow under strict timelines.
- **Asset Optimization:** Developed a comprehensive workflow for 3D model refinement in Maya and the integration of Spatial Audio to maximize environmental immersion.
